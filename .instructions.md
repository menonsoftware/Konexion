# VS Code Copilot Chat Instructions

## Purpose
This file provides rules and constraints for Copilot Chat to assist developers working on the Konexion project.

## Rules
1. **Backend**:
   - Use FastAPI patterns for route handlers.
   - Suggest async/await for all database and API calls.
2. **Frontend**:
   - Use kebab-case filenames for Svelte components.
   - Suggest writable stores for state management.
3. **General**:
   - Recommend `make` commands for setup and testing.
   - Validate pre-commit hook configurations.
